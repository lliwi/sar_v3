¿Qué incluye el backup?
  • Base de Datos Completa: Todos los datos del sistema en formato JSON
  • Esquema de BD: Estructura de tablas en formato SQL
  • Configuración: Archivo .env con todas las configuraciones
  • Metadatos: Información del backup y instrucciones de restauración

Casos de Uso
  • Recuperación ante Fallos: Restaurar el sistema después de un problema técnico
  • Migración de Servidor: Instalar el sistema en un nuevo servidor con todos los datos
  • Auditoría: Mantener copias históricas del estado del sistema
  • Desarrollo: Clonar el entorno de producción para testing
